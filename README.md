# Knapsack Program DSA JAVA
This program aims to measure the time executed using various algorithms 

# The algorithms :
* Abstraction
* Brute Force Algorithm
* Memoization Techniques ( Fastest )
* Recursive

# Data Structure Used 
* Map
* Hashmap
  
# Note
1. mainApp.java is the main CLI application
